说明

该网站用于分享个人的技术学习及面试准备的博客网站。为了管理及阅读更加方便，对个人博客进行了分类管理，分为技术学习、项目、前端基础、手写 API、后端、工具、算法几个章节，详情如下：

技术学习

该文件夹存放的是框架学习过程中的一些总结及使用比如 React 和 Vue

项目

该文件夹用于存放写项目过程中遇到的一些问题的解决方法及自己封装的一些工具的分享

前端基础

该文件夹用于存放 css 及 html 以及 js 的一些总结及相关八股文的学习分享

手写 API

该文件夹用于存放手写 API 的一些分享，比如手写 Promise 等，着重的提升自己对 api 的理解以及思维

后端

该文件夹用于存放学习的后端知识的总结，作为一个前端了解一些后端的知识也是必要的

工具

该文件夹用于存放前端工具的一些问题及总结比如说 git

算法

该文件夹用于存放算法刷题及学习过程中的总结
